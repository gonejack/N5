# 诺亚舟N5固件
诺亚舟N5固件v1.10

N5升级更新说明：
更新说明：此版本为4G版升级程序，2G版机器可以兼容；
==============================================================================

升级说明：
将升级文件maskrom.bin和PROGRAM.BIN拷贝到本地磁盘或者TF卡根目录下（建议使用TF卡拷贝）；

进入工具-系统设置-系统升级中，按照该目录中的升级操作说明进行升级；

==============================================================================

      注意事项：
      请保证在电量充足的情况下进行升级。
